EbitenPlus is a set of systems and prewritten code to help you develop games in ebitengine.

Features:

- Asset manager with asyncronous loading capabilities
- Ui components (buttons, forms, etc.)
- Camera with several different follow methods
- Graphics helpers (sprite objects, colors, vectors, etc.)

I will update this library when I want to and when I need to, feel free to use any of the code for personal projects.
